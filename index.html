燻製カマンベール味を使ったレシピ

食べた印象は、燻製感が強すぎず、クリーミーで優しい味わいだなと。そのまま食べても十分おいしいので、アレンジレシピはシンプルにしました。お酒のお供にはもちろん、ちょっとしたホームパーティーにもおススメです。

ホクホクじゃがいもと燻製味ベビーチーズ

じゃがいも、バター、チーズは鉄板の組み合わせ。燻製の香りに、ローズマリーをプラスして、風味を引き立てました。チーズと生ハムに塩気がるので、味付けは極力少なめに。でも、ブラックペッパーとケイパーがアクセントになっていて、味はしっかりしています。

詳しくはこちら

ホクホクじゃがいもと燻製味ベビーチーズ

調理時間：5分

調理ポイント：

材料

2人前

おうちDE居酒屋ベビー 燻製カマンベール味 4個
にんにく 1個
ローズマリー 1本
バター 40g
じゃがいも（メークイン） 2個
生ハム 6枚
赤ピーマン 適量
いんげん 適量
パセリ 適量
ケイパー 適量

作り方

  1. 1フライパンに軽くつぶしたにんにくとローズマリーを入れ、四つ切りにしたじゃがいもをバターでソテーする
  2. 2火が通ったら皿に取り出し、生ハム、小さくカットした赤ピーマン、湯がいたいんげんをのせ、パセリとケイパーをふりかける
  3. 3最後に、さいの目に切った「おうちDE居酒屋ベビー　燻製カマンベール味」をトッピングする

「ホームパーティ」をイメージした、“超”簡単にできて、見た目に華やかさのあるメニューにしました。燻製カマンベールがそのままでも十分おいしいので、味付けはシンプルにしています。
じゃがいもとチーズの組み合わせで子どもも大人も楽しめるおつまみです。じゃがいものソテーの代わりにフライドポテト（市販の冷凍ものなど）を代用してもおいしくいただけます。

おうちで美味しい燻製味チーズサンド

簡単・オシャレで、食感が楽しい一品。クロワッサンをクラッカーで代用すれば、おつまみ感がアップします。ベビーチーズの燻製感を味わえるよう、調味料を使わず食材の組み合わせでアレンジを。チーズで味がまとまるので、ドレッシングやマヨネーズの必要なしです。

詳しくはこちら

おうちで美味しい燻製味チーズサンド

調理時間：10分

調理ポイント：
チーズの厚さを揃えるようにしてください。パンは食パンでも可

材料

2人前

おうちDE居酒屋ベビー 燻製カマンベール味 4個
クロワッサン 2個
きゅうり 1/2本
マッシュルーム 2個
トマト 1/2個
スモークサーモン 2枚

作り方

  1. 1きゅうり、マッシュルーム、トマト、スモークサーモンを切り揃え、「おうちDE居酒屋ベビー　燻製カマンベール味」を半分の厚さにカットする
  2. 2切り込みを入れたクロワッサンに、1.の具材をきれいにサンドする
  3. 32.をお皿にのせる

ベビーチーズの燻製の風味を味わっていただくため、あえて調味料は使わずシンプルに仕上げています。
火も使わずカットしてサンドするのみの簡単なレシピで、食パンなど家にある他のパンでももちろんOK。
マッシュルームとチーズの風味の相性が非常によく、トマトときゅうりのみずみずしさとスモークサーモンの塩味がおいしさを引き立てます。

オーナーシェフ 相原 薫

プロフィール紹介

1974年神奈川県生まれ。1994年より神奈川県葉山のレストランで働き始める。2000年よりフランスやスイスで研鑽を重ね、ジュネーブの二つ星レストラン『ドメーヌ・ド・シャトーヴュー』では魚料理を担当するポワソニエを務めた。
2003年に帰国。『銀座レカン』、広尾『レヴェランス』、荻窪『ヴァリノール』などでシェフとして活躍し、2017年12月に「Simplicité（サンプリシテ）」を独立開業し、現在に至る。

焦がしにんにく＆ねぎ油風味
を使ったレシピ

ネーミング通り、ガツンとにんにくの香りがきて、ねぎ油の風味もしっかり。ねぎ油は作るのに手間がかかるので、その味がチーズで手軽に味わえるのはすばらしいです！この味わいをいかした、中国レシピを考えました。

あげ餃子焦がしにんにく＆
ねぎ油風味ベビーチーズ

中国でもポピュラーな揚げ餃子をアレンジした、ガッツリ系おつまみです。火を入れても溶けずに残るベビーチーズを丸ごと入れ、皮を折って包むのではなく、2枚使って円盤型に。ねぎ油でコクも出るので味付け必要なし！お好みで豆板醤を添えて、お酒に合う一品に。

詳しくはこちら

あげ餃子焦がしにんにく＆
ねぎ油風味ベビーチーズ

調理時間：20分

調理ポイント：
手軽におつまみ

材料

2人前

おうちDE居酒屋ベビー　焦がしにんにく＆ねぎ油風味 3個
豚バラ肉（スライス） 50g
パクチー 適量
餃子の皮 12枚
小麦粉 小さじ1
水 小さじ1
サラダ油 適量
豆板醬 少々

作り方

  1. 1餃子の皮を広げて、3cm幅にカットした豚バラ肉→半分の大きさにカットした「おうちDE居酒屋ベビー　焦がしにんにく＆ねぎ油風味」1/2個→パクチー1枚→豚バラ肉の順でサンドする
  2. 2小麦粉と水を混ぜ 1.の皮のまわりに塗り、さらに餃子の皮を被せる
    ※1.～2.を6個作る
  3. 3160℃の揚げ油で2分くらい揚げ、180℃でもう一度揚げる
  4. 43.を半分に切って盛りつけ、豆板醬を添える

家庭ではなかなか使うことはないけれど、中華では必須の調味料であるねぎ油の風味をいかした餃子です。
今回はあげ餃子にしましたが、もちろん焼き餃子でも。お酒のアテに最適です。
お好みで豆板醬と一緒に召し上がってください。

「吉田風中国家庭料理 ジーテン」
オーナーシェフ 吉田 勝彦

焦がしにんにく塩ラーメン
ねぎ油風味ベビーチーズ

呑み手のハートを鷲掴みする、〆料理としての一品を。やさしい味のチキンスープにコシのある麺、シャキシャキの具、そこに、ベビーチーズの焦がしにんにくとねぎ油の風味が重なり、口に入れた時に味が完成します！チーズは最後に入れて、絡めながら食べてください。

詳しくはこちら

焦がしにんにく塩ラーメン
ねぎ油風味ベビーチーズ

調理時間：15分

調理ポイント：
手軽に本格的な焦がしにんにくラーメンを。

材料

1人前

おうちDE居酒屋ベビー　焦がしにんにく＆ねぎ油風味 2個
鶏ひき肉 60g
もやし 50g
ニラ 20g
生麺 130g
サラダ油 大さじ1
チキンスープ 300cc
塩 小さじ1弱
日本酒 大さじ1
白だし 小さじ1
白コショウ 少々

作り方

  1. 1深いフライパンにサラダ油を入れて鶏ひき肉を弱火で炒めたら、チキンスープを加える
  2. 21.にもやし、3cmにカットしたニラ、＜調味料＞を入れる
  3. 3お湯を沸かしておいた鍋に麺を入れて茹でる
  4. 42.の具以外（スープ）を器に移し、3.の茹で上がった麺を入れる
  5. 52.の具を4.にのせ、「おうちDE居酒屋ベビー　焦がしにんにく＆ねぎ油風味」を手で割りながら散らしてトッピングする

〆に食べたくなるラーメンを考案。チーズを入れることでスープのコクが大幅アップします。
手軽な調理ながら、ベビーチーズの焦がしにんにく風味で本格的な味に仕上がります。

プロフィール紹介

1964年岩手県生まれ。高校卒業後、調理師学校の研修中に中国料理の賄いで夏バテが治った経験から、薬膳の勉強を始め中国料理の道へ。卒業後、埼玉県新座市の『四川飯店』へ入社し、その後23歳で代官山『Linka』へ。25歳で料理長になり6年間勤めた後、香港に渡り野菜中心の家庭料理を学ぶ。代々木上原『正宇治』の料理長を3年勤めた後、1999年代々木上原『吉田風中国家庭料理 ジーテン』をオーナーシェフとしてオープンし現在に至る。

カルボナーラ味を使ったレシピ

家族がベビーチーズ好きで、冷蔵庫にいろいろな種類があります。この「カルボナーラ味」はコクがあって、ベーコンとブラックペッパーの風味があとからくる感じです。今回はこのしっかりした味わいをいかしたレシピにしました。

ピエモンテ風玉葱グラタン

イタリア北西部・ピエモンテの郷土料理をアレンジ。生クリームを使わず、チーズのコクでグラタン風に。エキスを逃すことなく焼き上げた玉ねぎの驚きの甘さに、チーズの塩気と濃厚なうまみが加わって、食べ応え十分。ほんのりブラックペッパーで、あと引く味に！

詳しくはこちら

ピエモンテ風玉葱グラタン

調理時間：30〜60分

調理ポイント：
生クリームを使わない

材料

1人前

おうちDE居酒屋ベビー　カルボナーラ味 2個
玉ねぎ（新玉ねぎも可） 小さめ1個
市販のスープストック（顆粒） 適量
パン粉 適量
バター 適量

作り方

  1. 1玉ねぎを皮ごとアルミホイルに包み、180℃のオーブンで1時間加熱する（新玉ねぎの場合：加熱時間30分ほど）
  2. 21.の玉ねぎの上部を包丁かはさみで取り除き、スープストックを溶かしたお湯を流し込む
  3. 3小さめにカットした「おうちDE居酒屋ベビー　カルボナーラ味」を中に挟み入れ、トースターに入れる
    ※お好みで、トッピングにパン粉とバターを適量のせてトースターに入れてください
  4. 43.を加熱し、焼き色がついたら完成

別のベビーチーズを使ってもまた違う風味を楽しめる汎用性を意識しました。
こちらはピエモンテの郷土料理をベビーチーズでアレンジしたもの。生クリームを使わずチーズのコクでグラタン風に。
とても簡単なレシピなので忙しい女性向き、オーブンに入れている間に他の料理や作業の時間にあてられます。

「REGALO（レガーロ）」
オーナーシェフ 小倉 知巳

ローマ風カプレーゼ

ローマ料理のカルボナーラを、カプレーゼにアレンジ。ベーコンとブラックペッパーをプラスして加えることで、「カルボナーラ味」の味わいを、よりくっきりとさせています。おしゃれな見た目に反しパンチのきいた味は、スパークリング系のお酒と合わせても◎

詳しくはこちら

ローマ風カプレーゼ

調理時間：10分

調理ポイント：
チーズを手でちぎること、スライスした玉ねぎを塩揉みすること

材料

2人前

おうちDE居酒屋ベビー　カルボナーラ味 3個
ベーコン 2枚
玉ねぎ 少量
塩 ひとつまみ
フルーツトマト） 2個
ベビーリーフまたはバジル 適量
黒胡椒 適量
オリーブオイル 適量

作り方

  1. 1テフロンパンにベーコンを並べ、油をひかずに火をつけてじっくりカリカリにする
  2. 21.を作っている間に、薄切りにした玉ねぎをしんなりするまで塩揉みして、出てきた水分を捨てる
  3. 3ボウルに2.の玉ねぎ、一口大にカットしたフルーツトマト、ベビーリーフ、手でちぎった「おうちDE居酒屋ベビー　カルボナーラ味」を入れて混ぜ合わせる
  4. 43.をお皿に盛りつけて、1.のカリカリベーコンをあしらう
  5. 54.に黒胡椒をしっかり挽く
    ※お好みでオリーブオイルをかけてお召し上がりください

イタリアンの前菜メニュー定番のカプレーゼをベビーチーズならではの一品にアレンジ。
カリカリベーコンとしっかり挽いた黒胡椒が、ベビーチーズのカルボナーラ味をよりくっきりさせます。
